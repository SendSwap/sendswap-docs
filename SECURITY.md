# Security

If you believe you found a security issue related to SendSwap's public website or user-facing service flow, please report it privately.

## Report a security issue

Email: support@sendswap.io

Suggested subject line:

`[Security] Short description of the issue`

## What to include

Please include as much of the following as possible:

- affected page or URL
- time of discovery
- steps to reproduce
- screenshots, if relevant
- wallet address or order identifier only if needed for investigation

Do not send private keys, seed phrases, or exchange passwords.

## Phishing warning

To reduce phishing risk:

- always verify that you are using `https://sendswap.io`
- double-check destination and refund addresses carefully
- start with a small test transaction when using a new route
- rely only on official links published by SendSwap

## Scope

This repository is public and should not be used to post sensitive exploit details in the open before coordinated handling.
