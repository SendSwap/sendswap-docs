# SendSwap Docs

Public documentation for `SendSwap`, a privacy-first, non-custodial instant crypto exchange focused on wallet-to-wallet swaps.

This repository is intended for public-facing documentation and transparency materials. It does not contain SendSwap's core backend or private operational infrastructure.

## What SendSwap is

- Non-custodial instant crypto exchange
- Wallet-to-wallet swap flow
- No signup or account creation on SendSwap
- 2000+ supported assets
- Fixed and floating rate modes
- Privacy-coin friendly routes including XMR and ZEC

## How the flow works

1. Choose the asset pair.
2. Enter the destination wallet address.
3. Review the quoted rate and fees.
4. Send the deposit transaction from your own wallet.
5. Receive the payout directly in your destination wallet.

## Important scope note

SendSwap does not operate as a custodial account-based exchange. For standard swaps on SendSwap, users are not asked to create an account or complete KYC on the platform itself. Depending on the selected route and execution conditions, downstream swap or liquidity partners may apply operational, compliance, or risk-related checks outside SendSwap's direct control.
